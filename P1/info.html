HTML5 e CSS

Na criação deste website, sobre futebol, utilizei CSS para estilização de todas as páginas, header, nav, main, aside, footer, imagens, texto, títulos, animações e responsividade do site e dos mesmo. Utilizei CSS Grid para estilizar/formatar a estrutura dos 2 layouts que criei/utilizei, e Flex no display de alguns conteúdos. Foram também utilizados diversos ícons, animações, fonts de letras, uma imagem Wordlcloud, algumas imagens e vídeos para tornar o site mais dinâmico. O formulário criado, na forma de um quizz, sobre o mais diverso conteúdo do site envia as respostas dos utilizadores para um e-mail. Além, das técinas e propriedas já enunciadas, utilizei várias técnincas do HTML e CSS asseguir enunciadas/tabeladas.

R29: Técnicas usadas

Carateristicas Elementos/técnicas Exemplos
Layouts Usou-se o CSS grid e CSS flexbox para construção de layouts Criei 2 layouts, um para um Menu mais dinâmico e responsivo na página intro.html e outro nas outras páginas.
Responsividade Foram usadas media queries que adaptam o layout para PC consoante o tamanho da janela do browser. Todas as páginas são responsivas adaptando o tamanho do contéudo ao tamanho da janela, mais específico: letras do texto ficam verdes apartir de max-width: 800px e o texto desaparece apartir de max-width: 600px.
Seletores Foram usados vários selectores, descritos na tabela css: selectores css Utilizei em todas as páginas para definir a font o selector universal.
Etiquetas HTML5 main, header, article, footer, figure... Em todas as páginas foram usadas e formatados de acordo com a disposição do grid, numa página de uma equipa o menu está em nav, a imagem e texto no main, título no header e video no footer.
Animações Foram feitas animações usando keyframes que permitem deslocar imagens/icons. Ampliação das opções no 1º zoom, deslocação das bolas de futebol e do "logo" e de 1 linha de texto no index.
